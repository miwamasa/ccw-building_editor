## 6. 凡例

```
記号の意味:
🌡 = 温度センサー
💧 = 湿度センサー
📊 = 流量センサー
⚡ = 電力センサー
┌─┐ = センサー位置
└─┘

空調機器:
AHU  = 外調機 (Air Handling Unit)
VAV  = 変風量ユニット (Variable Air Volume)
VRF  = 可変冷媒流量システム (Variable Refrigerant Flow)

線の意味:
───▶ = 空気の流れ
════▶ = 冷水の流れ
····▶ = データの流れ
```

## 7. 省エネ最適化ポイント

### ポイント1: 外気冷房の活用
```
条件: 外気温 22.5°C < 還気温 24.5°C
     ↓
アクション: 外気導入量 30% → 50%に増加
     ↓
効果: 冷房負荷 約15%削減
```

### ポイント2: 冷水温度の最適化
```
従来: 冷水温度 7°C (固定)
     ↓
改善: 外気温 22.5°C → 冷水温度 8°C に変更
     ↓
効果: チラーCOP 10%向上
```

### ポイント3: VRFとセントラルの協調
```
全館空調 → 基本負荷(外気処理)を担当
     ↓
個別空調 → 局所的な温度調整を担当
     ↓
効果: 全体効率 約8%向上
```
